# Statistical Methods

## Descriptive Statistics
- Mean: Average value
- Median: Middle value (robust to outliers)
- Mode: Most frequent value
- Range: Max - Min
- Standard Deviation: Spread around mean

## Inferential Statistics
- t-test: Compare two means
- ANOVA: Compare multiple means
- Chi-square: Categorical relationships
- Regression: Predict outcome from predictors

## Common Distributions
- Normal (bell curve): Most natural phenomena
- Binomial: Yes/No outcomes
- Poisson: Count data

## Key Formulas
- Standard deviation: √(Σ(x-x̄)²/n)
- Correlation: Cov(X,Y)/(σX × σY)
- R-squared: Explained variance / Total variance
