## Do's and Don'ts

✅ DO: Research competitors' top hashtags
✅ DO: Rotate tags (same tags get shadow-banned)
✅ DO: Use location-based tags
❌ DON'T: Use banned/spam tags
❌ DON'T: Use unrelated tags (fake engagement)
❌ DON'T: Use >15 tags (looks desperate)
