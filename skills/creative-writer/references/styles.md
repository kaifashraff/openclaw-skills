# Writing Styles Guide

## Blog Writing
- Hook: "You know that feeling when..."
- Structure: Problem → Agitation → Solution
- Length: 800-1500 words
- Include subheadings every 300 words
- End with question to drive comments

## Brand Storytelling
- Origin story format: Challenge → Solution → Outcome
- Customer-centric (not brand-centric)
- Emotional trigger: pride, belonging, achievement
- Sensory details: what does it look/sound/feel like?

## Social Media Copy
- First line must stop scroll
- Include line breaks for mobile
- Emoji strategic, not excessive
- Clear CTA (comment, share, DM, click)

## Long-form Content
- Deep value (2000+ words)
- Comprehensive coverage
- Expert insights
- Data + story blend
- Multiple perspectives

## Script/Dialogue
- Distinct voice per character
- Subtext (what's unsaid)
- Natural rhythm
- Conflict in every scene
