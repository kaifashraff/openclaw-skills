# Flyer Designer

Create effective promotional flyers for R Company zari embroidery products.

## Design Principles

### Brand Colors (Always Use)
- Gold: #D4AF37 — Premium, heritage
- Cream: #FFF8E7 — Soft, elegant
- Maroon: #800020 — Rich, traditional

### Typography
- Headings: Cinzel (elegant, regal)
- Body: Playfair Display (readable, classic)
- Accent: Poppins (modern, clean)

## Flyer Types

### 1. Product Showcase Flyer
**Purpose:** Highlight new collection or product line
**Size:** 1080×1080px (Instagram), A4 (print)
**Elements:**
- Hero product image
- Collection name
- Key features (3-4 bullets)
- Price range
- CTA (WhatsApp/Call)
- Brand logo

### 2. Festival Sale Flyer
**Purpose:** Promote seasonal sale or festival offer
**Size:** 1080×1080px
**Elements:**
- Festival greeting/emoji
- Sale percentage (big, bold)
- Limited time (urgency)
- Product images (3-4)
- Discount code
- Valid dates
- CTA

### 3. Wholesale Catalog Flyer
**Purpose:** Attract boutique and bridal shop buyers
**Size:** A4 or 1080×1350px
**Elements:**
- "Wholesale Inquiry" header
- Product grid (6-9 products)
- Price range indication
- Minimum order info
- Contact details
- Trust badges (200+ buyers, Pan-India)

### 4. Event/Exhibition Flyer
**Purpose:** Announce presence at trade show or exhibition
**Size:** A5 or social media
**Elements:**
- Event name + logo
- Dates and venue
- "Visit Us" message
- Sample products
- Booth number
- QR code to WhatsApp

## Tools for Creation

- Canva (easy, templates available)
- Adobe Express (free tier)
- Figma (more control)
- Photoshop (advanced)

## Tips for WhatsApp Flyers

- File size: <500KB (WhatsApp limit)
- Format: PNG or JPEG
- Resolution: 1080px width minimum
- Text: Large enough to read on phone
- Contrast: High for readability
