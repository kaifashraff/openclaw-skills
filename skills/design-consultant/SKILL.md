# Design Consultant

Transform ideas into visual solutions.

## Design Principles

1. **Hierarchy** — Most important elements stand out
2. **Contrast** — Create visual interest and clarity
3. **Alignment** — Organized, intentional placement
4. **Repetition** — Consistent elements throughout
5. **Proximity** — Related items grouped together

## Color Theory

### Brand Palette (R Company)
- Gold #D4AF37 — Premium, heritage, warmth
- Cream #FFF8E7 — Soft, elegant, background
- Maroon #800020 — Rich, traditional, accent

### Color Relationships
- Complementary: 180° apart (contrast)
- Analogous: 30° apart (harmonious)
- Triadic: 120° apart (vibrant)

## Typography

| Use | Font | Style |
|-----|------|-------|
| Headlines | Cinzel | Elegant, regal |
| Body | Playfair Display | Classic, readable |
| UI/Accent | Poppins | Modern, clean |

### Hierarchy
- H1: 48-64px (titles)
- H2: 32-40px (section)
- H3: 24-28px (subsection)
- Body: 16-18px
- Caption: 12-14px

## Layout Principles
- 8pt grid system
- White space = breathing room
- Visual balance
- Mobile-first design

## Feedback Framework

When critiquing design:
1. What works well? (Be specific)
2. What could be improved?
3. Suggest alternatives (not just complaints)
