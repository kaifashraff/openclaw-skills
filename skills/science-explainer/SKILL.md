# Science Explainer

Make complex science accessible and engaging.

## Communication Principles

1. **Start with the "why"** — Why does this matter?
2. **Build from known** — Connect to familiar concepts
3. **Concrete, not abstract** — Use analogies
4. **Limit jargon** — Define technical terms
5. **Visualize when possible** — Mental models help

## Approach by Complexity

### Simple (Core concept)
- One key idea
- Analogy or example
- Takeaway message

### Moderate (Multi-part)
- Background context
- Key components explained
- How they connect
- Real-world application

### Complex (Deep dive)
- Layered explanation
- Evidence and data
- Uncertainty acknowledged
- Implications discussed

## Common Topics

- Physics: Forces, energy, quantum
- Biology: Evolution, genetics, ecology
- Chemistry: Reactions, materials
- Technology: AI, space, computing
- Health: Medicine, nutrition, exercise

## Addressing Misconceptions

1. Acknowledge the misconception
2. Explain why it's intuitive but wrong
3. Present the correct model
4. Give evidence
5. Connect back to everyday life

## Science vs Pseudoscience

| Science | Pseudoscience |
|---------|---------------|
| Falsifiable | Not testable |
| Peer review | No review |
| Uncertainty acknowledged | False certainty |
| Updates with evidence | Stagnant claims |
