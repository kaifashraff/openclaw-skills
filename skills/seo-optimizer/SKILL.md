# SEO Optimizer

Optimize Laqab v2 and R Company online presence for search engines.

## On-Page SEO

### Meta Tags (Per Page)

**Homepage:**
```html
<title>R Company — Premium Handcrafted Zari Embroidery | Ahmedabad, India</title>
<meta name="description" content="Discover exquisite handcrafted zari embroidery, sherwani, kurta sets, and bridal wear. 200+ verified buyers trust R Company. Pan-India shipping. Custom designs available.">
<meta name="keywords" content="zari embroidery, handcrafted ethnic wear, sherwani, bridal wear, indian ethnic fashion, wholesale ethnic clothes, ahmedabad">
```

**Product Pages:**
```html
<title>[Product Name] | Premium Handcrafted Zari | R Company</title>
<meta name="description" content="[2-sentence product description focusing on key features and benefits. Include price range and customization options.]">
```

### Image Optimization

- Use descriptive filenames: `handcrafted-zari-sherwani-gold.jpg` not `IMG_001.jpg`
- Alt text required for every image
- Compress images (target <200KB)
- Use WebP format where possible
- Lazy load below-fold images

## Technical SEO

### Site Speed (Target: <3s load time)

1. Minimize HTTP requests
2. Enable browser caching
3. Use CDN for assets
4. Optimize CSS/JS
5. Image compression
6. Gzip compression

### Mobile Optimization

- Responsive design (Laqab v2 already has)
- Touch-friendly buttons (min 44px)
- Fast mobile page speed
- No intrusive popups

## Local SEO (Ahmedabad)

### Google Business Profile
- Claim and verify listing
- Use correct categories: "Embroidery Shop", "Ethnic Clothing Store"
- Add photos regularly
- Respond to reviews
- Update hours during festivals

### Local Keywords
- "zari embroidery ahmedabad"
- "ethnic wear shop near me"
- "wholesale ethnic clothes gujarat"
- "sherwani ahmedabad"
