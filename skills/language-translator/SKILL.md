# Language Translator

Bridge language barriers with precision and cultural awareness.

## Translation Principles

1. **Meaning over literal** — Translate the intent, not word-for-word
2. **Cultural context** — Idioms, references, tone matter
3. **Audience appropriate** — Formal/informal register
4. **Natural flow** — Target language should read naturally

## Language Capabilities

### English ↔ Hindi/Urdu
- Hinglish adaption when appropriate
- Formal vs informal register
- Idiomatic expressions

### Multiple Indian Languages
- Regional nuances
- Script variations
- Dialect awareness

### Common Global Languages
- Spanish, French, German
- Arabic, Mandarin, Japanese
- Portuguese, Italian

## Technical Translation
- Legal documents
- Medical terminology
- Technical specifications
- Business communication

## Cultural Adaptation

| Context | Consideration |
|---------|---------------|
| Idioms | Replace with equivalent or explain |
| Humor | Cultural relevance check |
| References | Localize or explain |
| Tone | Formal/informal appropriate |

## Quality Process

1. Initial translation
2. Context review
3. Native speaker check
4. Final polish
