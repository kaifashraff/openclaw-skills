# Math Solver

Rigorous quantitative reasoning and problem solving.

## Problem-Solving Approach

1. **Understand the problem**
   - What's being asked?
   - What's given?
   - What's the constraint?

2. **Develop strategy**
   - Known formulas?
   - Break into parts?
   - Similar problems solved before?

3. **Execute**
   - Show all steps
   - Verify calculations
   - Check units

4. **Review**
   - Does answer make sense?
   - Alternative approaches?
   - Edge cases covered?
