# Retargeting Setup Guides

## Google Ads Retargeting

### Audience Setup
1. Google Ads → Audience Manager
2. Create "Website visitors" audience
3. Select: All visitors, last 30 days
4. Create nested audiences:
   - Cart abandoners (specific pages)
   - Product viewers (specific products)
5. Set membership duration: 30-90 days

### Dynamic Remarketing
1. Link Google Merchant Center
2. Upload product feed
3. Create dynamic display ads
4. Set bidding to "Target ROAS"

## Facebook/Instagram Retargeting

### Pixel Setup
1. Facebook Events Manager
2. Install pixel on Laqab v2
3. Track events: PageView, ViewContent, AddToCart, InitiateCheckout, Purchase

### Audiences
1. Create "Website visitors" (all, 30 days)
2. Create "ViewContent" (specific products)
3. Create "AddToCart" (cart abandoners)
4. Create lookalike from "Purchase" converters

### Budget Recommendations
| Audience | Daily Budget | Duration |
|----------|--------------|----------|
| Cart abandoners | ₹200-300 | Until converted |
| ViewContent | ₹150-200 | 14 days |
| All visitors | ₹100-150 | 30 days |

## WhatsApp Retargeting

### For Non-Converters
1. Export WhatsApp Business leads
2. Create targeted broadcast lists
3. Send follow-up sequence:
   - Day 0: "Hope you had a chance to think about [Product]"
   - Day 3: "Still available! Quick question?"
   - Day 7: "Last chance" with special offer

### For Past Visitors
1. Tag visitors by interest
2. Share relevant products
3. Offer exclusive discount
4. Include "Reply YES to order" option
