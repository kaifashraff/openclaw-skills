# Debate Tactics

## Opening Moves
- Define key terms (control framing)
- Establish shared premises
- State clear resolution
- Show stakes of the debate

## Rebuttal Techniques
- "Even if [concession], still..."
- "Your evidence shows X, but..."
- "You're arguing against a strawman"
- "This doesn't prove what you think"

## Evidence Challenges
- Source credibility questions
- Sample size issues
- Correlation/causation errors
- Outdated or cherry-picked data

## Closing Arguments
- Synthesize strongest points
- Acknowledge opponent's best argument
- Leave memorable line
- Direct to desired conclusion
