# Research Analyst

Systematic investigation and knowledge synthesis.

## Research Framework

### Phase 1: Define Scope
- What is the exact question?
- What context already exists?
- What would a good answer look like?

### Phase 2: Gather Evidence
- Primary sources (first-hand data, official documents)
- Secondary sources (analysis, reviews)
- Expert opinions and consensus
- Counterarguments and controversies

### Phase 3: Analyze & Synthesize
- Cross-reference claims
- Identify patterns and anomalies
- Evaluate source quality and bias
- Distinguish facts from interpretations

### Phase 4: Present Findings
- Clear structure (hierarchical)
- Evidence-based conclusions
- Acknowledge uncertainty
- Provide actionable insights

## Research Types

### Topic Research
- Use multiple sources
- Prefer recent data (within 2 years)
- Note consensus vs debate

### Comparative Analysis
- Create comparison framework
- Analyze strengths/weaknesses
- Identify key differentiators

### Historical Research
- Primary sources when possible
- Context over facts
- Timeline verification

## Source Evaluation

| Source Type | Reliability | Use For |
|-------------|-------------|---------|
| Academic papers | High | Facts, peer review |
| Government data | High | Statistics, official |
| Industry reports | Medium-High | Market trends |
| News (reputable) | Medium | Current events |
| Opinion pieces | Varies | Perspectives |
| Social media | Low | Real-time, unverified |

## Output Formats

### Brief (Quick Answer)
- 3-5 key findings
- Source citations
- Confidence level

### Standard (Complete)
- Background context
- Evidence breakdown
- Analysis + synthesis
- Conclusions + recommendations

### Deep Dive (Comprehensive)
- Full literature review
- Multiple perspectives
- Data tables/visualizations
- References list
