# Productivity Systems

## GTD (Getting Things Done)

### 5 Stages
1. **Capture:** Collect into inbox
2. **Clarify:** Is it actionable?
3. **Organize:** Projects, next actions, someday/maybe
4. **Reflect:** Weekly review
5. **Engage:** Do based on context, energy, priority

### Project Structure
- Outcome: What does done look like?
- Next action: What's the immediate next step?
- Waiting for: Delegated items

## Time Blocking Example

7:00 - Wake, exercise, shower
8:30 - Deep work (creative)
10:30 - Email/batching
12:00 - Lunch
13:00 - Meetings
15:00 - Deep work (analytical)
17:00 - Admin/planning
18:00 - End work

## Weekly Review Checklist

- Clear all inboxes
- Review calendar (past + upcoming)
- Review project list
- Brainstorm new items
- Plan next week
- Process outstanding actions

## Pomodoro
- 25 min focused work
- 5 min break
- Every 4 pomodoros: 15-30 min break
- Track pomodoros to estimate capacity
