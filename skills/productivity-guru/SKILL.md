# Productivity Guru

Work smarter, not just harder.

## Productivity Principles

1. **Prioritize ruthlessly** — Not all tasks are equal
2. **Protect your time** — Your most valuable resource
3. **Energy management** — Rest is part of work
4. **Systems over willpower** — Design for success
5. **Progress over perfection** — Done beats perfect

## Time Management

### Eisenhower Matrix
| | Urgent | Not Urgent |
|---|---|---|
| **Important** | Do immediately | Schedule |
| **Not Important** | Delegate | Eliminate |

### Deep Work
- 2-4 hour blocks of uninterrupted time
- Single task focus (no multitasking)
- Phone on airplane mode
- Notifications off
- Create before consume

### Time Blocking
- Calendar = commitment
- Time-box tasks
- Buffer time between blocks
- Protect blocks fiercely

## Focus & Distraction

### When Distracted
- Urgent ≠ Important
- Ask: "Will this matter in 5 years?"
- Batch similar tasks
- Browser: Use blockers (Cold Turkey, Freedom)

### Procrastination
- Start with 5 minutes (overcome inertia)
- Break into smallest possible step
- Remove friction to starting
- Accountability helps

## Habit Building

### Habit Loop
1. Cue: Trigger that starts habit
2. Routine: The habit itself
3. Reward: Benefit you get
4. Repeat: Make automatic

### Building New Habits
- Stack onto existing habit
- Make it easy (reduce friction)
- Start tiny (2-minute version)
- Track consistently
- Reward yourself

### Breaking Bad Habits
- Remove the cue
- Add friction
- Make the reward less accessible
- Replace with better alternative

## Energy Management

### High Performance Habits
- Sleep 7-8 hours
- Move daily
- Eat real food
- Morning routine
- Strategic rest breaks

### Rest Types
- Physical: Sleep, exercise
- Mental: Walking, nature
- Emotional: Connection, leisure
- Creative: Hobbies, inspiration

## Productivity Systems

- Getting Things Done (GTD): Capture, clarify, organize, reflect, engage
- Time blocking: Calendar-based planning
- MITs: 3 Most Important Tasks daily
- 90-day goals: Quarterly sprints
