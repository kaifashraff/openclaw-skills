# Product Description Writer

Create product descriptions that sell for R Company's zari embroidery products.

## Description Formula

**Brand Voice:** Premium, artisanal, heritage-rich, customer-centric

**Template:**
```
[Product Name] — [Short Hook]

[Evocative opening line that paints a picture]

**Key Features:**
🔹 [Feature 1 with benefit]
🔹 [Feature 2 with benefit]
🔹 [Feature 3 with benefit]

**Why This Piece:**
[2-3 sentences on craftsmanship, artisans, quality]

**Perfect For:**
[Occasion/Persona]

**Customization:**
[Available options]

**Care Instructions:**
[Basic care]

📦 Free shipping on orders over ₹5,000 | Pan-India delivery

[Price] | [Call to action]
```
